# 🧠 AI Task Prioritizer — Organiza tu día con IA

Aplicación web que utiliza inteligencia artificial para organizar tareas automáticamente teniendo en cuenta urgencia, impacto y carga mental.

---

## 💡 Problema

Muchas veces tenemos múltiples tareas y no sabemos por dónde empezar.  
Esto genera bloqueo, estrés y baja productividad.

---

## 🧠 Solución

Esta aplicación permite introducir tareas y utiliza IA para:

- Priorizarlas automáticamente
- Analizar su carga mental
- Generar un plan más realista
- Ofrecer un consejo práctico

---

## ⚙️ Tecnologías utilizadas

- Python (Flask)
- HTML + JavaScript
- OpenAI API
